# Instruções do projeto

Este é o portfólio pessoal do Raul, hospedado no GitHub Pages.

## Fluxo de trabalho
- Depois de qualquer alteração aprovada nos arquivos, faça o commit
  automaticamente com uma mensagem descritiva em português.
- Em seguida, dê push para o branch principal (origin).
- Não é necessário pedir confirmação extra para o commit/push — a aprovação
  do diff da alteração já serve como aprovação.
